## üìÅ Results Directory (Local Artifacts)

The `results/` directory contains **locally generated artifacts** produced during
training and analysis. Due to their size and reproducibility, **large artifacts
such as trained model weights and bulk Grad-CAM visualizations are intentionally
excluded from version control** via `.gitignore`.

These files can be regenerated by following the execution steps outlined below.

### Typical Local Contents (Not Tracked in Git)

- baseline_cnn_model.h5 ‚Äî Trained baseline CNN model
- resnet50_tl_model.h5 ‚Äî Trained ResNet50 transfer learning model
- baseline_metrics.txt ‚Äî Evaluation metrics for baseline model
- class_names.json ‚Äî Class label mapping
- test_samples.json ‚Äî Fixed test samples for Grad-CAM analysis

- Dataset distribution visualizations
